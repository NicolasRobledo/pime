# Stored Procedures - Purchases Module

Stored procedures relacionados con la gestión de compras y ventas.

## Stored Procedures sugeridos:

- `sp_create_purchase.sql` - Registrar nueva compra
- `sp_get_purchase_by_id.sql` - Obtener compra por ID
- `sp_list_user_purchases.sql` - Listar compras de un usuario
- `sp_list_all_purchases.sql` - Listar todas las compras (admin)
- `sp_update_drive_shared_status.sql` - Marcar archivo como compartido en Drive
- `sp_get_purchases_by_date_range.sql` - Obtener compras por rango de fechas
- `sp_get_sales_stats.sql` - Estadísticas de ventas
