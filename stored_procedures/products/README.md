# Stored Procedures - Products Module

Stored procedures relacionados con la gestión de productos (modelos 3D).

## Stored Procedures sugeridos:

- `sp_create_product.sql` - Crear nuevo producto
- `sp_update_product.sql` - Actualizar producto existente
- `sp_get_product_by_id.sql` - Obtener producto por ID
- `sp_list_products.sql` - Listar productos con filtros (anime, personaje, categoría)
- `sp_delete_product.sql` - Eliminar producto
- `sp_update_mercadolibre_id.sql` - Actualizar ID de MercadoLibre tras publicación
- `sp_toggle_product_status.sql` - Activar/desactivar producto
